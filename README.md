Bash script to auto install Arch Linux  

### Installation
Clone the repo.
```
git clone https://github.com/Atsika/arch.git
cd arch
```  

### Usage
```
chmod +x install.sh
./install.sh
```

### Description
The script creates a user **arch** with password **arch**.  
Also it sets **root** as password for **root** account.  
You better change those.
